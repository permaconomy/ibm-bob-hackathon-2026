# 🚀 Deploy Your Website to IBM Cloud - Super Simple Guide

## What You Need Before Starting

1. ✅ Your website files (you already have these!)
2. ✅ An IBM Cloud account (free) - Sign up at https://cloud.ibm.com
3. ✅ IBM Cloud CLI installed on your computer

---

## Step 1: Install IBM Cloud CLI (5 minutes)

### For Windows:

1. Go to: https://cloud.ibm.com/docs/cli?topic=cli-install-ibmcloud-cli
2. Click "Download for Windows"
3. Run the installer (just click "Next" through everything)
4. Open PowerShell and type: `ibmcloud version`
   - If you see a version number, it worked! ✅

---

## Step 2: Login to IBM Cloud (2 minutes)

Open PowerShell and type these commands:

```powershell
# Login to IBM Cloud
ibmcloud login

# It will ask for:
# - Your email (the one you used to sign up)
# - Your password
# Just type them and press Enter
```

**If it asks about regions:** Choose the one closest to you (like "us-south" or "eu-gb")

---

## Step 3: Set Up Cloud Foundry (1 minute)

Still in PowerShell, type:

```powershell
# This tells IBM Cloud you want to use Cloud Foundry
ibmcloud target --cf

# It will show you organizations and spaces
# Just pick the first one by typing "1" and pressing Enter
```

---

## Step 4: Go to Your Website Folder (1 minute)

In PowerShell, type:

```powershell
cd "c:\Users\DELL\Desktop\IBM Bob Dev Day Hackathon\compliance-web-interface"
```

**Check you're in the right place:** Type `dir` and you should see files like `app.py`, `requirements.txt`, etc.

---

## Step 5: Deploy Your Website! (3-5 minutes)

This is the magic command! Type:

```powershell
ibmcloud cf push my-compliance-checker
```

**What happens now:**
- IBM Cloud uploads your files ⬆️
- Installs Python and all needed packages 📦
- Starts your website 🌐
- Gives you a web address (URL) 🔗

**Wait for it to finish!** You'll see lots of text scrolling. When it stops and shows "App started", you're done! ✅

---

## Step 6: Get Your Website Address

After deployment finishes, look for a line that says:

```
routes: my-compliance-checker.mybluemix.net
```

**That's your website address!** 🎉

Your full URL will be: `https://my-compliance-checker.mybluemix.net`

---

## Step 7: Share with Your Team

Copy your website URL and send it to your team members:

```
Hey team! Our website is live at:
https://my-compliance-checker.mybluemix.net

You can now upload files and test the compliance checker!
```

## 🎯 Quick Commands You Might Need

### See if your website is running:
```powershell
ibmcloud cf apps
```

### See what's happening (logs):
```powershell
ibmcloud cf logs my-compliance-checker --recent
```

### Restart your website:
```powershell
ibmcloud cf restart my-compliance-checker
```

### Update your website (after making changes):
```powershell
# Just run the deploy command again!
ibmcloud cf push my-compliance-checker
```

---

## ❌ If Something Goes Wrong

### Problem: "Command not found"
**Fix:** IBM Cloud CLI isn't installed. Go back to Step 1.

### Problem: "Login failed"
**Fix:** Check your email and password. Make sure you signed up at cloud.ibm.com first.

### Problem: "No space targeted"
**Fix:** Run this command:
```powershell
ibmcloud target --cf
```

### Problem: "App name already taken"
**Fix:** Use a different name:
```powershell
ibmcloud cf push my-compliance-checker-team5
```

### Problem: Website shows error page
**Fix:** Check the logs:
```powershell
ibmcloud cf logs my-compliance-checker --recent
```
